CS111 Introduction to Computer Science 1

This course is a rigorous introduction to computer science that develops computations problem solving skills by programming in Python. This course covers a variety of introductory computer science lessons from setting variables to using dictionaries and object oriented programming to analyze pieces of literature.

This course covers advanced programming techniques and data structures. Topics include recursion, algorithm analysis, iterators, linked lists, stacks, queues, trees, graphs, hashtables, searching, sorting.
